# 摸鱼阅读 P0需求技术文档（总索引）

## 使用方法（省 token 版）
- 先确定你要查的对象：`P0编号`、`PRD章节`、`验收指标` 三选一。
- 再用关键字检索，不要整篇通读。

## 快速检索命令
```bash
# 1) 按 P0 编号查（如 P0-08）
rg -n "P0-08" docs/p0

# 2) 按 PRD 章节查（如 2.3 / 3.4）
rg -n "2.3|3.4" docs/p0

# 3) 按页面/模块查（如 目录/老板键/导入）
rg -n "目录|老板键|导入|进度|阅读核心" docs/p0

# 4) 按验收指标查（如 < 0.1 秒 / 60fps）
rg -n "0.1 秒|60fps|150MB|导入耗时" docs/p0

# 5) 查工1实施文档（P0-04/05/06/07）
rg -n "P0-04|P0-05|P0-06|P0-07|实施清单|导入新书" docs/p0
```

## 推荐阅读顺序（按任务）
- 做功能：先查 `P0-xx` 映射，再看对应技术设计段落。
- 做联调：先查路由与数据结构，再看该条目验收标准。
- 做测试：直接查“验收与测试”中的指标和用例编号。
- 做工1交接（P0-04/05/06/07）：先读 `docs/p0/03-实施清单_工1_P0-04到P0-07.md`。

## 维护规则
- PRD变更：先改映射，再改实现。
- 功能改动：只改对应条目，不做整文档重写。
